🎭 Playwright Automation Framework Documentation

📌 Overview

This framework is built using Playwright with JavaScript/TypeScript to enable fast, reliable, and cross-browser automation. It supports Chromium, Firefox, and WebKit, and integrates with tools like Allure, GitHub Actions, and Docker for scalable execution.

🏗️ Architecture

🔹 Folder Structure

playwright-framework/
│
├── tests/               # Test scripts
├── pages/               # Page Object Models
├── utils/               # Reusable functions
├── config/              # Environment configs
├── reports/             # Allure or HTML reports
├── node_modules/        # Dependencies
├── playwright.config.ts # Global config
└── package.json         # Project metadata

🔹 Design Pattern

  • Page Object Model (POM)
  • Modular structure
  • Config-driven execution

⚙️ Tools & Technologies

Tool Purpose
Playwright Web automation
TypeScript Strong typing and scalability
Allure Report Rich test reporting
GitHub Actions CI/CD pipeline
Docker Containerized execution

🧩 Key Components

  • Test Scripts: Located in /tests
  • Page Objects: Encapsulate selectors and actions in /pages
  • Utilities: Common helpers in /utils
  • Configuration: playwright.config.ts and environment files

🧪 Test Data Strategy

Use JSON or CSV files stored in /config/testdata, loaded dynamically via utility functions.

📊 Reporting

  • Allure Reports with screenshots and step logs
  • Optional: Playwright’s built-in HTML report

🚀 Execution Modes

Local

npx playwright test

Headless

npx playwright test --headed=false

CI/CD

GitHub Actions workflow and Dockerfile for containerized runs

🔐 Environment Setup

Prerequisites

  • Node.js 18+
  • Playwright installed via npm

Setup Steps

git clone https://github.com/milanmore/playwright-framework
cd playwright-framework
npm install
npx playwright install

🧠 Troubleshooting & FAQs

  • Issue: Browser not launching
    Fix: Run npx playwright install
  • Issue: Timeout errors
    Fix: Increase timeout in playwright.config.ts
