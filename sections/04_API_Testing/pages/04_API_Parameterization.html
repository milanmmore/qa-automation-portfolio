Introduction

Parameterization enables dynamic input values during test execution, allowing the same test logic to run against multiple data sets. Here's how to implement it using popular tools and formats:


Using Postman for Parameterization
  • Supported Data Formats:
    • CSV: Comma-separated values with headers as variable names
    • JSON: Array of objects with key-value pairs
  • Steps to Implement:
    1. Prepare your data file:

      CSV Example:
      username,password
      user1@example.com,pass123
      user2@example.com,pass456

      JSON Example:
      [
        { "username": "user1@example.com", "password": "pass123" },
        { "username": "user2@example.com", "password": "pass456" }
      ]
    2. Use variables in your request:
      Replace static values with dynamic variables using double curly braces:
      { "email": "{{username}}", "password": "{{password}}" }
    3. Run the collection:
      Use Postman's Collection Runner to upload the data file and execute tests.
  • Accessing Variables in Scripts:
    You can reference data variables in Postman test scripts using the data object:
    pm.test("Status code is 200", function () {
      pm.expect(pm.response.code).to.eql(200);
      console.log("Testing with:", data.username);
    });
  • Note:
    Excel files must be converted to CSV before use in Postman.

Using TypeScript with Playwright
  • Supported Data Formats:
    • JSON: Directly import and iterate over objects
    • CSV: Use libraries like csv-parse or papaparse
    • Excel: Use xlsx to read and convert sheets
  • JSON Example:
    View Code 
    
            import test from '@playwright/test';
            const testData = require('./data.json');
            
            testData.forEach(({ username, password }) => {
              test(`Login test for ${username}`, async ({ page }) => {
                await page.goto('https://example.com/login');
                await page.fill('#email', username);
                await page.fill('#password', password);
                await page.click('button[type="submit"]');
                await page.waitForSelector('#dashboard');
              });
            });
  • CSV Example:
    View Code 
    
            import fs from 'fs';
            import parse from 'csv-parse/lib/sync';
            
            const csvData = fs.readFileSync('./data.csv');
            const records = parse(csvData, { columns: true });
            
            records.forEach(({ username, password }) => {
              test(`Login test for ${username}`, async ({ page }) => {
                // Same login steps
              });
            });
  • Excel Example:
    View Code 
    
            import * as XLSX from 'xlsx';
            
            const workbook = XLSX.readFile('./data.xlsx');
            const sheet = workbook.Sheets[workbook.SheetNames[0]];
            const data = XLSX.utils.sheet_to_json(sheet);
            
            data.forEach(({ username, password }) => {
              test(`Login test for ${username}`, async ({ page }) => {
                // Same login steps
              });
            });
