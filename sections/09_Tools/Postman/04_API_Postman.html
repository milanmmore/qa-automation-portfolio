üîß Write Test Scripts

In the Tests tab, use JavaScript to validate responses:



                pm.test("Status code is 200", () => {
                pm.response.to.have.status(200);
                });
                
                pm.test("Response has user data", () => {
                const jsonData = pm.response.json();
                pm.expect(jsonData.data).to.have.property("email");
                });

                pm.test("Check if user is created", function () {
                    pm.response.to.have.status(201);
                    pm.response.to.be.json;
                    pm.expect(pm.response.json().data).to.have.property("id");
                });

You can chain multiple assertions and use Chai-style syntax.

üîß Use Collections for Modular Testing

  • Attach pre-request scripts and tests to each request
