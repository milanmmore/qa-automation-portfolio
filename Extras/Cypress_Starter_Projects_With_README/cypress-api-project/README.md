# Cypress API Testing Starter Project

## Setup Instructions
1. Install dependencies:
   ```bash
   npm install
   ```

2. Run API tests:
   ```bash
   npx cypress run --spec cypress/e2e/api_test.cy.js
   ```

3. Open Cypress Test Runner (optional):
   ```bash
   npx cypress open
   ```

## Files
- `cypress/e2e/api_test.cy.js`: Example API tests using reqres.in
