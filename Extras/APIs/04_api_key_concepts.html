  • Endpoints: URLs where APIs are accessed (e.g., GET /users/{id})
  • HTTP Methods: GET, POST, PUT, DELETE for CRUD operations
  • Request Payloads & Parameters: Query strings and JSON bodies
  • Responses & Status Codes: Validate 200 OK, 400 Bad Request, etc.
  • Headers: Metadata like Authorization and Content-Type
  • Assertions: Check status codes, response body, and performance
  • Authentication & Authorization: API keys, OAuth2, JWTs
  • Performance & Reliability: Load, stress, and spike testing
